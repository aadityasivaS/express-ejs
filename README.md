## Getting Started

To get started follow the instruction below

### Installing

To install run

```
npm i -g express-ejs
```

### Usage

To start run
```
express-ejs your-project-name
```

This will clone all the files and install all packages

To start the server run

```
npm start
```

This will start the server at port 3000
